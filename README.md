# Оно тебе надо

Аукцион вещей, в которые никто не верил

## Описание

Верстка лендинга для аукциона необычных вещей. Проект включает в себя:

- Главную страницу с героем-секцией
- Галерею лотов
- Информацию об аукционе
- Адаптивную верстку

## Технологии

- HTML5
- CSS3
- Flexbox/Grid
- Шрифты Raleway

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ваш-username/ono-tebe-nado-ad.git
```

2. Откройте `index.html` в браузере
